	Import Question & Answer form DW Question & Answer plugin.

== Description ==

Simlesly import question and answer data from DW Question & Answer plugin, 


* Comments
* Best answer.
* Question vote
* Answer vote
* Categories
* Tags

<strong>How to import</strong>

If you want import on same website where DWQA also installed you can directly import through following menu "Question Answer - DW Migration"

If you want move DWQA from another site, you will need to import xml first by using default import & export menu. and then run "Question Answer - DW Migration"

<strong>Translation</strong>

Pluign is transaltion ready , please find the 'en.po' for default transaltion file under 'languages' folder and add your own translation. you can also contribute in translation, please contact us http://www.pickplugins.com/contact/
